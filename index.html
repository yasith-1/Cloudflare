☁️ Cloudflare යනු කුමක්ද?

Internet ලෝකයේ "Security Guard" ගැන ඔබ දැනගත යුතු සෑම දෙයක්ම — සරල බසින්, ජීවිතේ උදාහරණ සමඟ.

ඔබ "Cloudflare" කියන වචනය ඇහුවාම... "ඒ මොකක්ද?" කියලා හිතෙනවා ඇති. අද ඒ ගැන ඔබ දැනගත යුතු සෑම දෙයක්ම — ළමයෙකුටත් තේරෙන සරල භාෂාවෙන් කියා දෙන්නම්.

🏫 පළමුව — Cloudflare කියන්නේ කොහොම එකක්ද?

Cloudflare යනු 2009 දී ඇමරිකාවේ ආරම්භ කළ Internet Infrastructure Company එකක්. ඔවුන්ගේ ප්‍රධාන කාර්යය තමයි Internet ලෝකය ආරක්ෂා කරන එක සහ වේගවත් කරන එක.

🍎 ජීවිතේ උදාහරණය

ඔයාගේ ගෙදර ගේට්ටුව ළඟ Security Guard කෙනෙක් ඉන්නවා කියලා හිතන්න. ඒ Guard කෙනා බලනවා — ගෙදරට එන අය ඇත්තටම ඔයාලාගේ හිතවතුන්ද, නැත්නම් හොරෙක්ද කියලා. හොරෙක් ආවොත් ඒ Guard කෙනා එයාව block කරනවා. හිතවතෙකු ආවොත් ඉක්මනින් ඇතුළට ගන්නවා. Cloudflare කරන්නේ අන්න ඒ වැඩේමයි — හැබැයි Websites වලට!

📖 Cloudflare කරන ප්‍රධානතම වැඩෙන් එකක් — DNS

ඔබ browser එකේ google.com කියලා type කරනකොට — computer එක හොයනවා "google.com තියෙන්නේ කොහෙද?" කියලා. DNS (Domain Name System) කියන්නේ ඒකට අදාළ address book එකක් වගේ දෙයක්.

📞 Phone Book උදාහරණය

ඔයාට "රාජ්" කියන කෙනාගේ ගෙදරට යන්න ඕනෑ නම් ඔයා phone book එක බලනවා — එතකොට රාජ්ගේ address එක හොයාගන්න පුළුවන්. DNS කියන්නේ අන්න ඒ Phone Book එක වගේ. google.com = 142.250.80.46 (IP address) කියලා DNS එකෙන් කියනවා.

Cloudflare ලෝකයේ වේගවත්ම DNS 1.1.1.1 ලබා දෙනවා — ඒක සම්පූර්ණයෙන්ම FREE සහ Private, ඒ වගේම ඔබේ දත්ත track කරන්නේ නැතුව (No tracking)!

🌍 CDN — Website එක ඉක්මන් කරන Magic එක

CDN කියන්නේ Content Delivery Network කියන එකයි. Website වල copies ලෝකේ පුරා තියෙන servers දහස් ගණනක (නගර 300+ ක) store කරලා — ඔයාට ළඟම තියෙන server එක හරහා website එක deliver කරනවා.

🍕 Pizza Shop උදාහරණය

Pizza shop එකක් Colombo වල විතරක් තිබ්බොත් — Galle ඉඳලා order කළොත් delivery එකට ගොඩක් time යනවා. ඉක්මනින් දෙන්න නම් Galle, Kandy, Jaffna වගේ නගර වලත් branches තියෙන්න ඕනේ. CDN කියන්නේ අන්න ඒ branches වලට. ඔයාට ළඟම තියෙන branch එකෙන් deliver කරනකොට website එක තත්පර ගාණකින් load වෙනවා!

🛡️ DDoS Attack කියන්නේ මොකක්ද? Cloudflare ආරක්ෂා කරන්නේ කෙසේද?

DDoS (Distributed Denial of Service) Attack එකක් කියන්නේ — Hackers ලා website එකට එකවර fake requests මිලියන ගණනක් යවන එකටයි — එතකොට Server එක crash වෙලා real users ලාට site එකට access වෙන්න බැරි වෙනවා.

🏦 Bank Queue උදාහරණය

Bank එකක පෝලිම පුරාවටම fake customers ලා පිරුණොත් — ඇත්ත customers ලාට service ගන්න බැරි වෙනවා. Bank එකේ වැඩ අඩාල වෙනවා. DDoS Attack එකකුත් ඒ වගේ. Cloudflare ඒ fake requests ටික filter කරලා real users ලාට විතරක් service දෙනවා — තත්පරයකට Attacks මිලියන ගණනක් Block කරනවා!

📊 ඇත්ත Data

Cloudflare දිනකට අන්තර්ජාලයේ traffic එකෙන් 20% ක් පමණ handle කරනවා. ලෝකේ පුරා Internet users ලා බිලියන 5 කට වඩා ඉන්නවා — Cloudflare ඒ ගොඩදෙනෙක්ගේ requests තත්පර ගාණක් ඇතුළත process කරනවා.

🔒 Web Application Firewall (WAF)

WAF කියන්නේ Website එකේ ඉන්න "Smart Bouncer" වගේ කෙනෙක්. Hackers ලා website එක hack කරන්නට try කළොත් — SQL Injection, XSS Attacks, Bot Attacks වගේ දේවල් — Cloudflare WAF හරහා ඒ දේවල් filter කරලා block කරනවා.
