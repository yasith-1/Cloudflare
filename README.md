## 📋 Overview (මොකක්ද මේ?)

The **Cloudflare සම්පූර්ණ Guide** is a high-fidelity technical article designed to simplify the world of Web Infrastructure and Security for Sinhala-speaking users. Using relatable real-world analogies (Pizza shops, Bank queues, and Security guards), this guide demystifies complex network concepts like CDNs, DDoS protection, and WAFs.

### ✨ Key Features (ප්‍රධාන අංග)

- 📡 **DNS & Speed Logic** - Explains how the world's fastest 1.1.1.1 DNS works with privacy.
- 🛡️ **Edge Security Simulation** - Visualizes Cloudflare's WAF and DDoS protection through real-world scenarios.
- 🍕 **CDN (Pizza Shop Logic)** - Demonstrates how Edge caching reduces latency across 300+ cities.
- 📊 **Performance Analytics** - Insight into how requests are processed at the Edge.
- ⏱️ **Plan Comparison** - Clear assessment of Free vs Paid plans for small businesses and developers.
- ⚙️ **Step-by-Step Setup** - A simple 4-step walkthrough from account creation to Nameserver updates.
- 🤖 **Future Tech Layers** - Introduction to Workers, R2 Storage, Tunneling, and Zero Trust.

</br>
